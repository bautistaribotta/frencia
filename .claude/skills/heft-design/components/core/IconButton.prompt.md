Icon-only button for toolbars, steppers and nav controls.

```jsx
<IconButton icon="plus" variant="primary" round />
```

Variants: `primary` (emerald), `surface` (chip), `ghost`. Sizes `sm`/`md`/`lg` — use `md`+ for touch (44px). Set `round` for circular. Pass a Lucide name to `icon` and run `lucide.createIcons()` after mount.
