Compact mono status marker for set type, RIR/RPE state, or a PR flag.

```jsx
<Badge tone="orange-solid" icon="flame">Nuevo PR</Badge>
<Badge tone="green">Completada</Badge>
```

Tones: `neutral`, `green`, `green-solid`, `orange`, `orange-solid`, `danger`. Reserve solid tones for emphatic moments; soft tones for inline state. Always uppercase mono.
