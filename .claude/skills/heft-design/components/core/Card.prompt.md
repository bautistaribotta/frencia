Surface container that groups related content on the dark field.

```jsx
<Card variant="green" hairline>
  <span className="heft-label">RIR medio</span>
  <div style={{ font: 'var(--text-data)' }}>1.8</div>
</Card>
```

Variants: `default`, `elevated` (shadow — floating only), `inset` (data well), `green`/`orange` (tinted accent state). Add `hairline` for a subtle border, `interactive` for tappable cards.
